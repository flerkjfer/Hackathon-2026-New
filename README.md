# React CRUD Starter

A small React + Vite starter that demonstrates CRUD actions with data saved in browser `localStorage`.

## What it does

- Create records with a name, description, and status
- Read records in a responsive card list
- Update existing records
- Delete records
- Persist data between refreshes with `localStorage`

## Run it

1. Install Node.js if it is not already available on your machine.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

## Put it on GitHub

After you create a GitHub repo, connect this folder to it:

```bash
git init
git add .
git commit -m "Initial React CRUD starter"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```
